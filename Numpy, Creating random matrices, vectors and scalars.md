```py
import numpy as np

# Create a random scalar
scalar_rand = np.random.randint(5, 15)
print(scalar_rand)

# Create a vector with rand nums from 5-15
arr_rand = np.random.randint(5, 15, 4)
print(arr_rand)

# Create a 5x5 matrix with rand nums from 5-15
matrix_rand = np.random.randint(5, 15, (5, 5))
print(matrix_rand)
```
